# bravo-vince

B R A V O V I N C E your text in Node.js.

See also: [bravovince.com](http://bravovince.com)

# Usage

Install:

```
yarn add bravo-vince
```

Use:

```js
import bravoVince from 'bravo-vince'

const output = bravoVince('bravo', ['vince', 'avocado'])
console.log(output)
```

Output:

```
B
R
A V O C A D O
V I N C E
O
```
